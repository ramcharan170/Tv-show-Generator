# TV Show Generator

A Netflix-style TV show search application built using JavaScript and Axios.

## Features
- Search TV shows dynamically
- Fetch data from TVMaze API
- Horizontal Netflix-style movie layout
- Responsive UI
- Dynamic DOM rendering
- Async/Await implementation

## Technologies Used
- HTML
- CSS
- JavaScript
- Axios
- Bulma CSS

## Concepts Learned
- API Fetching
- Async/Await
- DOM Manipulation
- Dynamic Rendering
- Flexbox Layout
- Horizontal Scrolling UI

## API Used
https://api.tvmaze.com/
